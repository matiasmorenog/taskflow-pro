# TaskFlow Pro

TaskFlow Pro is a task management app focused on real delivery workflows. It demonstrates React state architecture, reusable UI patterns, filtering logic, and local persistence.

## Features

- Create tasks with title, description, priority and due date
- Update task status (`Todo`, `In Progress`, `Done`)
- Quick toggle for completion
- Delete tasks and clear completed ones
- Search by title/description
- Filter by status and priority
- Sort by `Newest`, `Oldest`, or `Priority`
- Dashboard stats with completion rate
- LocalStorage persistence

## Tech Stack

- React
- Vite
- CSS (custom responsive UI)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
